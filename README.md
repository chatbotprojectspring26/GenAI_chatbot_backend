# GenAI_chatbot_backend

FastAPI backend for GenAI Research Chatbot Platform with A/B testing capabilities.

## 🎯 Features

- **A/B Testing System**: Random assignment for research studies
- **Multi-LLM Integration**: OpenAI + Binghamton LLAMA API
- **Session Management**: Complete participant and conversation tracking
- **Real-time Logging**: Comprehensive session monitoring
- **RESTful API**: 15+ endpoints for research operations

## 🏗️ Architecture

- **Framework**: FastAPI with SQLModel ORM
- **Database**: SQLite (development) with MongoDB migration path
- **LLM**: OpenAI GPT-4o-mini with automatic fallback
- **Testing**: Database-level random assignment for A/B tests
